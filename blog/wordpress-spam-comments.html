Wordpress spam comments

Marc | Fri 29 June 2007

Today I've installed a new WordPress plugin, to avoid robots write comments on this page. It's very common that spammers use blog comments to link their pages, as publicity, and for increasing their pagerank.

It can be solved in WordPress using JSSpamBlock plugin from Paul Butler. Thanks man!
